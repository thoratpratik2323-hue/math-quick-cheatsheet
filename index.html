Last Minute Revision Hacks for Ishu's 80/80 Target

Standard Substitutions

  • $\sqrt{a^2 - x^2}$ : Put $x = a\sin\theta$
  • $\sqrt{a^2 + x^2}$ : Put $x = a\tan\theta$
  • $\sqrt{x^2 - a^2}$ : Put $x = a\sec\theta$
  • $\sqrt{\frac{a-x}{a+x}}$ : Put $x = a\cos2\theta$
Substitution is the key to solving 90% of Inverse Trigo and Calculus long questions!

Calculus Trigo Life-Savers

  • $1 - \cos 2x = 2\sin^2 x$
  • $1 + \cos 2x = 2\cos^2 x$
  • $\sin 2x = 2\sin x \cos x = \frac{2\tan x}{1+\tan^2 x}$
  • $\cos 2x = \cos^2 x - \sin^2 x = \frac{1-\tan^2 x}{1+\tan^2 x}$
CRITICAL

Don't forget: $\sin^3 x = \frac{3\sin x - \sin 3x}{4}$

Matrix & Det Hacks

  • $|kA| = k^n|A|$ ($n$ = order)
  • $|adj A| = |A|^{n-1}$
  • $|A \cdot adj A| = |A|^n$
  • Skew-symmetric matrix of ODD order always has determinant = 0.
$A^{-1} = \frac{1}{|A|} adj A$

Vector Super-Fast

  • Projection of $\vec{a}$ on $\vec{b} = \frac{\vec{a} \cdot \vec{b}}{|\vec{b}|}$
  • Unit vector perpendicular to BOTH $\vec{a}$ and $\vec{b} = \pm \frac{\vec{a} \times \vec{b}}{|\vec{a} \times \vec{b}|}$
  • If $\vec{a}, \vec{b}, \vec{c}$ are coplanar $\implies [\vec{a} \vec{b} \vec{c}] = 0$.

3D Shortcut

100% Asked

  • Distance of point $(x_1, y_1, z_1)$ from plane $ax+by+cz+d=0$:
    • $d = \frac{|ax_1 + by_1 + cz_1 + d|}{\sqrt{a^2+b^2+c^2}}$
  • Angle between 2 planes = Angle between their normals.

Probability Logic

  • $P(A \cup B) = P(A) + P(B) - P(A \cap B)$
  • Independent Events: $P(A \cap B) = P(A) \cdot P(B)$
  • $P(A|B) + P(A'|B) = 1$
If events are mutually exclusive, $P(A \cap B) = 0$.

Limit & Diff Hacks

  • $\frac{d}{dx}(\sqrt{x}) = \frac{1}{2\sqrt{x}}$
  • $\frac{d}{dx}(|x|) = \frac{x}{|x|}$ (for $x \neq 0$)
  • $\frac{d}{dx}(\log_a x) = \frac{1}{x \ln a}$
If $y = \sqrt{f(x) + \sqrt{f(x) + \dots}}$, then $\frac{dy}{dx} = \frac{f'(x)}{2y-1}$.

Relation Counting

  • Total relations on set $A$(n) = $2^{n^2}$
  • Total Reflexive relations = $2^{n^2 - n}$
  • Total Symmetric relations = $2^{n(n+1)/2}$
